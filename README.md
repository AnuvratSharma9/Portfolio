Uber Rides Data Analysis
Overview

Explored over 150,000 Uber rides to understand how distance, pricing, payment methods, and ratings affect overall business performance.
Used SQL for cleaning and querying, and Python (Pandas, Matplotlib, Seaborn) for data analysis and visualization.
Focused on identifying ride demand trends, revenue patterns, and customer behavior insights.

Key Steps
Cleaned and structured raw data by removing duplicates and handling missing values.
Converted date fields into month, day, and hour for better time-based analysis.
Engineered new features like fare per kilometer to compare ride efficiency.
Conducted exploratory analysis to study trends across ride type, payment mode, and distance.

Skills & Tools Used

Languages: SQL, Python
Libraries: Pandas, Matplotlib, Seaborn,Numpy,pymysql
Concepts: Data Cleaning, Feature Engineering, Exploratory Data Analysis, Visualization

Outcome

Gained a clear understanding of what drives demand, pricing, and satisfaction in ride-sharing data.
Strengthened analytical skills by combining SQL querying with Python-based EDA.
Demonstrated ability to extract business insights from large real-world datasets.
